Projeto - Solução Questionário Socioeconômico

Do que se trata esta página?

Está página e um modelo ou protótipo que pode se tornar um Sistema para facilitação da análise socioeconômica dos estudantes da Fatec Franca.

Porque foi feito em web?

A aplicação foi desenvolvida em web utilizando JS (JavaScript) pois é uma forma mais simples para a utilização, já que não necessita de download ou instalação de componentes extras.
